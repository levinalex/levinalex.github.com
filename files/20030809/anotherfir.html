Image replacement

This is a slightly changed variant of the Fahrner Image Replacement technique.. It adresses a critical flaw that makes the content unusable if images are turned off in a graphical browser. Sadly, it still requires additional markup to achieve the effect.

Example:

Headline

The HTML:

<h1 class="replace" id="myh1">Headline<span></span></h1>

The CSS:

Works in:

  • Opera 5 and up (thanks Jay Meinhard)
  • IE 6
  • Mozilla Firebird
  • Mozilla 1.2
  • Konqueror 3.1.2
  • Safari
  • IE5.2/Mac OS X (thanks Philippe)

Fails in:

  • IE 5.1.3/Mac OS 9.2; (please verify)

Last Changes:

I removed the overflow and z-index rules and moved the <span> to the end. I split the styles into a class and an id to improve reusability. Browser support should not have changed
